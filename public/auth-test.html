Common Firebase Issues

If you're seeing 400 errors, check:

  1. Firebase Console: Go to Firebase Console
  2. Authentication: Enable Email/Password authentication
  3. Firestore: Create a Firestore database (if not already created)
  4. Authorized Domains: Add localhost to authorized domains in Authentication settings
  5. API Keys: Ensure API keys are not restricted for localhost
